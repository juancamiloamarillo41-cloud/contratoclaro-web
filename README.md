# ContratoClaro

**ContratoClaro** es una aplicación Android que analiza contratos y los explica en lenguaje claro mediante inteligencia artificial. El producto fue diseñado, desarrollado y publicado de forma individual en un ciclo de aproximadamente dos meses.

[![Google Play](https://img.shields.io/badge/Disponible_en-Google_Play-01875F?style=for-the-badge&logo=googleplay&logoColor=white)](https://play.google.com/store/apps/details?id=com.contratoclaro)
[![Sitio web](https://img.shields.io/badge/Ver-sitio_web-1E5BC6?style=for-the-badge)](https://juancamiloamarillo41-cloud.github.io/contratoclaro-web/)

## Qué resuelve

- Resume contratos extensos en lenguaje comprensible.
- Identifica cláusulas que requieren atención y explica su posible impacto.
- Genera análisis con contexto legal mediante una arquitectura RAG.
- Permite consultar el documento, guardar el historial y generar informes.
- Integra funciones premium mediante suscripciones de Google Play.

## Arquitectura del producto

| Área | Tecnologías |
|---|---|
| Aplicación móvil | Kotlin, Android y Jetpack Compose |
| Inteligencia artificial | Google Gemini y procesamiento de lenguaje natural |
| Recuperación de contexto | RAG sobre Vertex AI |
| Datos y autenticación | Firebase y Firestore |
| Monetización | Google Play Billing |
| Colaboración y versiones | Git y GitHub |

## Sobre este repositorio

Este repositorio contiene la landing pública de ContratoClaro, desplegada mediante GitHub Pages. El código fuente de la aplicación móvil se mantiene privado porque corresponde a un producto publicado y en evolución.
